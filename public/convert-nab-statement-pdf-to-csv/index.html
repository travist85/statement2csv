BankSheet

Convert NAB Statement PDF to CSV

Need transaction data from an NAB (National Australia Bank) statement PDF? BankSheet lets you upload the statement, review extracted transactions, and export a clean CSV in minutes.

Best for

  • Older statement periods where only PDF is available
  • Archived or closed accounts
  • Client-provided PDFs for bookkeeping
  • Reducing manual copy/paste and cleanup

How it works

  1. Upload your NAB statement PDF
  2. Review extracted transactions in the preview table
  3. Download CSV (or XLSX if preferred)

Privacy

Statement files and transaction data are deleted after conversion.

FAQ

Does this work on all NAB statement formats?

It works best on standard text-based PDFs. Scanned/image-only PDFs may be less reliable.

Can I use the CSV for accounting workflows?

Yes. CSV is ideal for spreadsheet cleanup and many import workflows. Bank-specific schemas can be handled as a later step.

What if a line is missing or parsed incorrectly?

Use the issue link and include what statement format failed. Those examples directly improve parser reliability.
